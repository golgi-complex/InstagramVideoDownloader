# Instagram Video Downloader + Processor

---

## 🇷🇺 Русский

### 📌 Описание

Это приложение позволяет:
- скачивать видео из Instagram по ссылкам
- автоматически обрабатывать видео (фон + размытие + затемнение)
- изменять формат видео под горизонтальный (1920x1080)
- применять визуальные эффекты к фону
- сохранять результат в выбранную папку

---

### ⚙️ Возможности

- 📥 Загрузка видео через `yt-dlp`
- 🎞 Обработка видео через `OpenCV`
- 🌫 Настраиваемое размытие фона
- 🌑 Настраиваемое затемнение фона
- 📊 Прогресс-бар с процентами
- 🌐 Поддержка нескольких языков (RU / EN)
- 📂 Выбор папки сохранения
- 🧵 Многопоточная загрузка (UI не зависает)

---

### 🧩 Технологии

- Python 3
- yt-dlp
- OpenCV (cv2)
- Tkinter
- NumPy
